jobs-feed
===
jobs feed slider

## install
#### tools
`npm install -g json-server`
`npm install -g grunt-cli`

#### dependencies
`npm install`
`bower install`

## run
#### server
`json-server data/db.json`

#### project
`grunt base`
`grunt build`
`grunt pack` - optional
open `app/index.html`

## tools
Atom.io (+linter)
Terminal
Safari
